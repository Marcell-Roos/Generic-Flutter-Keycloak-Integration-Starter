# Generic Flutter Keycloak Integration Starter

This Flutter project is a starter template to integrate your Keycloak instance with a flutter application,
it is aimed to be simple to use and work on all flutter supported platforms.

## Installation

You will need to already have a Keycloak instance configured, other than that you can clone this project and
add your Keycloak url, realm name and client id to the constants.dart folder.
